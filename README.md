# Python Project
這是一個儲存 Python 各個小專案的地方。

## 安裝模組
> **pip 一行安裝**
```bash
pip install beautifulsoup4 mediapipe numpy opencv-python Pillow pyautogui Pygments pylatexenc pymongo PyQt5 requests selenium
```
> **pip 文件安裝**
```bash
pip install -r requirement.txt
```
